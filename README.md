# Learning_Android_app_test_automation
**Prerequisite**

	Java
	Ecplise IDE
	Android studios
	Appium Server
	UI Automator
	TestNG plugin

**How to execute this code**

Install all the prerequist.
Find the file "TestSuite_testng.xml" > right-click & click on "Run > TestNG Suite"
